# Documentação

Esta pasta concentra os documentos de planejamento do projeto.

## Arquivos

- `proposta.MD`: proposta inicial de "Drecut: Fragmentos da Verdade", com conceito, integrantes, regras, controles, escopo mínimo e melhorias planejadas.

## Relação com o projeto atual

A implementação atual segue a ideia principal da proposta: um jogo de mistério em que o detetive conversa com NPCs, resolve desafios e precisa evitar a perda de vítimas.

## Uso recomendado

- Atualizar a proposta quando houver mudança de escopo.
- Registrar decisões importantes de design e implementação.
- Manter uma lista clara do que já foi implementado e do que ainda falta para a entrega final.
